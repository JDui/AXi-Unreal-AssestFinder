# Release Notes

## v1.2.1 - Directory Repair UX And Reference Map

发布日期：2026-06-29

### 新增

- 目录修复模式改为药丸滑块切换。
- 引用文件按目录层级渲染为脑图风格结构，并根据与当前资产目录的距离调整亮度。
- 目录修复拖入文件夹和检测路径时显示代码流扫过动画。
- README 更新为无弹窗静默目录修复界面截图。

### 改进

- 修复目录修复检测结果双击定位时可能跳转不到正确文件的问题。
- 分析结果选中文本使用亮绿底、深色字，避免自定义前景色覆盖选区反色。
- 统一目录修复“新目录名”输入框的禁用态和选区配色。
- 目录修复“执行变更”按钮禁用态改为暗色，降低误判为可点击的风险。
- 删除 README 许可证段落。

### 构建产物

- `UEAssetPathFinder-v1.2.1-win64.zip`

## v1.2.0 - Directory Safety And Matrix Chrome

发布日期：2026-06-29

### 新增

- 目录变更支持新旧目录名长度不一致。
- 新旧目录名长度不一致时增加二次确认。
- 切换到“原位更改模式”时提示“该操作可能不可逆，注意备份！”。
- 执行目录变更前提示目录外引用丢失风险。
- 自绘 Matrix 风格标题栏、弹窗和棱角化窗口边缘。
- 重绘 AX Matrix 软件图标，并同步 GUI 图标和 Windows EXE 图标。
- README 更新为静默主界面截图。

### 改进

- 默认窗口高度增加到 820，宽度保持 1000。
- 下拉选择列表、输入框、边框和弹窗统一为 Matrix 视觉风格。
- 分析结果和检测结果侧边滑条统一为 Matrix 视觉风格。
- 分析结果和全局文本选区使用亮绿底、深色字，选中文本颜色反转。
- 保持拖放区呼吸提示和资产解析代码流扫过动画。

### 构建产物

- `UEAssetPathFinder-v1.2.0-win64.zip`

## v1.1.0 - Matrix UI Refresh

发布日期：2026-06-29

### 新增

- 深色矩阵风格界面。
- 顶部扫描条动画。
- 拖拽区域霓虹边框脉冲。
- 资产解析时整页代码流扫过动画，最短显示 0.5 秒。
- 终端风结果面板、输入框和状态栏。
- README 软件界面截图展示。
- `DEVELOPMENT.md` 开发、测试、构建、发布流程文档。

### 改进

- 重写 `README.md`，补充安装、运行、构建、限制和目录修复说明。
- 重写 `IMPLEMENTATION.md`，记录核心模块和业务流程。
- 重绘标签页，选中态更大更亮，未选中态保持稳定尺寸。
- 重绘按钮、输入框、面板边框、表格、进度条和状态栏样式。
- 文本选区统一使用亮绿底和深色字，提升选中时可读性。
- 目录模式下拉选择始终可点击，仅执行按钮在校验通过前禁用。

### 保持不变

- 资产解析逻辑不变。
- 拖拽处理逻辑不变。
- 目录校验和目录变更逻辑不变。
- PyInstaller 打包入口不变。
