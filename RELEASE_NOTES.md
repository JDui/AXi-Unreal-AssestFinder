# UE Asset Path Finder - 发布说明

## 📦 版本信息
- **名称**: UE Asset Path Finder (UE资产路径查找器)
- **版本**: 1.0 Release
- **发布日期**: 2024
- **编译方式**: PyInstaller 6.18.0 (单文件模式)
- **目标平台**: Windows 64-bit

## 🎯 功能特性

### 核心功能
- ✅ **UE资产分析** - 自动识别UE资产文件（.uasset、.umap等）
- ✅ **版本检测** - 准确识别资产所属的虚幻引擎版本（UE 4.x / UE 5.x）
- ✅ **类型识别** - 自动检测资产类型（Blueprint、Mesh、Material、Texture等）
- ✅ **拖拽支持** - 直接拖拽文件到窗口进行分析
- ✅ **文件浏览** - 集成文件/目录浏览器
- ✅ **一键复制** - 快速复制UE内路径到剪贴板

### 界面设计
- 现代化 Material Design 色彩方案
- 中文字体优化（微软雅黑）
- 直观的 Emoji 图标标识
- 清晰的分组信息展示

## 📋 资产信息输出

分析资产时会显示以下信息：

| 分类 | 内容 | 说明 |
|-----|------|------|
| 📍 UE内路径 | `/Game/...` | 资产在UE Content文件夹中的相对路径 |
| 📄 基本信息 | 文件名、大小、类型 | 资产的基础属性 |
| 🎮 引擎信息 | UE版本、版本代码、魔数 | 引擎相关的技术信息 |
| 📁 关联文件 | .uexp、.ubulk等 | 同名的关联二进制文件 |
| 🔗 导入/引用 | 前20条导入记录 | 资产引用的其他资产信息 |

## 🚀 使用方法

### 方式一：拖拽分析（推荐）
1. 启动 `UEAssetPathFinder.exe`
2. 从文件资源管理器中拖拽UE资产文件到窗口
3. 在结果框中查看分析结果
4. 点击 "复制路径" 按钮复制UE内路径到剪贴板

### 方式二：文件浏览
1. 启动 `UEAssetPathFinder.exe`
2. 点击 "📁 浏览文件" 按钮
3. 选择要分析的UE资产文件
4. 查看分析结果

### 方式三：目录扫描
1. 启动 `UEAssetPathFinder.exe`
2. 点击 "📂 浏览目录" 按钮
3. 选择包含UE资产的目录
4. 自动扫描并列出目录中的所有资产

## 📊 技术规格

### 支持的文件类型
- `.uasset` - 虚幻资产文件
- `.umap` - 虚幻地图文件
- `.uexp` - 虚幻导出文件（关联）
- `.ubulk` - 虚幻块数据文件（关联）

### 检测能力

#### UE版本检测
- **UE 4.0-4.9** - 准确识别小版本号
- **UE 5.0-5.1** - 特殊版本区间识别
- **UE 5.2+** - 最新版本识别

#### 资产类型检测（优先级顺序）
1. Blueprint_C - 蓝图
2. SkeletalMesh - 骨骼网格
3. StaticMesh - 静态网格
4. Material - 材质
5. Texture2D - 纹理
6. AnimSequence - 动画序列
7. ParticleSystem - 粒子系统
8. DataTable - 数据表
9. 更多自定义类型...

### 系统要求
- **操作系统**: Windows 7 或更高版本（推荐 Windows 10/11）
- **处理器**: Intel/AMD x86-64 兼容处理器
- **内存**: 最低 512MB
- **磁盘**: 20MB 空闲空间（应用程序 ~11MB）

## 🔧 技术细节

### 核心技术栈
- **Python 3.12.6** - 编程语言
- **Tkinter** - GUI框架
- **TkinterDnD2 0.4.0** - 拖拽功能支持
- **PyInstaller 6.18.0** - 打包编译工具

### 架构说明
```
UEAssetPathFinder.exe (单文件可执行)
├── Python 3.12 运行时
├── Tkinter GUI库
├── TkinterDnD2 拖拽扩展
├── 资产解析器 (ue_parser)
├── 路径转换工具 (asset_finder)
└── 配置系统 (config)
```

### 二进制解析
- 使用 `struct` 模块逐字节解析UE资产文件
- 精确提取：版本号、魔数、导出表、导入表
- 支持多种数据类型和编码格式
